# Quantum Sand OpenFashionLabel

Quantum Sand OpenFashionLabel is a Blender app.

In a nutshell, Blender apps are “Experiences Powered by Blender”.

The file format is JSON and is based around the concept of parametric design.

This will include;
* 2d patterns to create garments
* 3d geometry for rendering real-time cloth in Blender
* resources textures for both 2d cloth printing and 3d rendering in Blender

```json
{
  "name": "ofl-example-project",
  "version": "0.1.0",
  "patternmaking": {},
  "geometry": {},
  "resources": {},
  "license": "CC-BY-SA-4.0"
}
 ```
